OpenNPCX is an organization which specializes in reverse engineering & open sourcing the NPCX line of chips.

Currently being worked on:
- NPCX7M6F

## Projects
*At the time of writing, some projects may not be publically accessible yet.*
- [website](https://github.com/OpenNPCX/website) - Our website hosted at [opennpcx.kxtz.dev](https://opennpcx.kxtz.dev)
- [dumpfw](https://github.com/OpenNPCX/dumpfw) - A minimal Chrome EC fork made to get the EC on as fast as possible for it to be ready to dump memory (no AP bringup, no GPIO, etc, etc)
- [rom](https://github.com/OpenNPCX/rom) - Decompilations of the NCPX roms. Each chip has it's own branch. For example: `NPCX7M6F` chip series -> `rom-npcx7m6f` branch
- [emulator](https://github.com/OpenNPCX/emulator) - Emulators for the NPCX chips. Each chip has it's own branch. Example: `NPCX7M6F` chip series -> `emu-npcx7m6f`  branch.
